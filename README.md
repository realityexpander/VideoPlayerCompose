# VideoPlayerCompose
Shows how to use the Exoplayer Video Player library inside compose

- Allows user to playback web or locally recorded videos
- Allows user to record videos

## Tech used
- Kotlin
- Hilt
- Compose
- Exoplayer in an AndroidView
- Coroutines
- Alert Dialogs
- File System access
